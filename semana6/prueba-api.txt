Al enviar una petición POST a http://localhost:3000/registro con el siguiente JSON:
{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor responde con:
{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Básicamente el servidor recibe los datos, los lee con req.body y los devuelve confirmando que llegaron correctamente.
