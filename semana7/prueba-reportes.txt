Al enviar una petición POST a http://localhost:3000/reportes con este JSON:
{
  "tipo": "Infraestructura",
  "descripcion": "Daño en alumbrado público"
}

El servidor responde confirmando que el reporte fue registrado:
{
  "mensaje": "Reporte registrado",
  "reporte": {
    "id": 1,
    "tipo": "Infraestructura",
    "descripcion": "Daño en alumbrado público"
  }
}

Al consultar GET http://localhost:3000/reportes el servidor devuelve la lista con todos los reportes registrados.
Si no se ha enviado ninguno devuelve un array vacío, si se enviaron varios los muestra todos en orden.
